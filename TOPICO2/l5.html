Partícula DE

A partícula で serve para apontar o objeto no qual você vai utilizar para fazer determinada ação. Por exemplo: (beber água utilizando o copo, comer utilizando colher).

Outro uso para partícula で é relatar onde alguma ação acontece. Por exemplo: (ler na biblioteca, jogar bola na quadra, comer no restaurante).

Você vai levar sua amiga para sair sem avisar o local que a levará, assim ela pergunta: “onde vamos?”, você responde: “vamos juntos ver um filme no cinema”. Vamos ver como montar esta frase a seguir.

いっしょにえいがかんでえいがをみます。
Vamos juntos assistir um filme no cinema. (で = local que ocorre a ação)


としょかんでほんをよみます。
Vou ler livros na biblioteca. (で = local que ocorre a ação)


こうえんであたしいぼうしをかいます。
Vou comprar um novo chapéu no parque (で = local que ocorre a ação)


このいとをハサミできります。
Vou cortar este fio com tesoura (で = utilizando a tesoura para realizar a ação do verbo (cortar))


けいたいでおにいさんとはなします。
Vou conversar com meu irmão usando o celular. ( で = utilizando o celular para realizar a ação do verbo(conversar))


ペンでぶんしょうをかく。
Escrever a redação com caneta. で = utilizando a caneta para realizar a ação do verbo(escrever))

Perguntas Simples


Quando formamos perguntas no português, basta que acrescentemos o ponto de interrogação (?). No japonês também funciona assim, mas só em situações informais.


Para formar perguntas com o desu basta que se acrescente o ka (em hiragana) logo depois.

KA é uma partícula de fim de frase usada para tornar o que foi dito em pergunta.

Quando formamos frases usando o ka, não é preciso colocar o ponto de interrogação.

Note que aqui essa partícula deve acompanhar o desu, usá-lo diretamente tornaria a frase rude.

Formal:

べんとう　です　か。✅

Informal:

べんとう　か。❌
べんとう　？✅

Assim sendo, podemos formar frases como

きょうですか。
kyou desu ka
É hoje?


あしたですか。
ashita desu ka
É amanhã?


ほんですか。
hon desu ka
É um livro?


みずですか。
mizu desu ka
É água?


とりですか。
tori desu ka
É um passáro?


まんがですか。
manga desu ka.
É um mangá?


きつねですか。
kitsune desu ka.
É uma raposa?


つきか。
tsuki ka.
É a lua?


としょかんか。
toshokan ka.
É uma biblioteca?


ゆびか。
yubi ka.
É um dedo?
